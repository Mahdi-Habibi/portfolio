# 21st.dev

This project uses the unified **[@21st-dev/cli](https://www.npmjs.com/package/@21st-dev/cli)** (`21st`) for searching, installing, and generating UI components from [21st.dev](https://21st.dev).

## What was installed

| Item | Location |
|------|----------|
| CLI package | `app/frontend` (devDependency `@21st-dev/cli`) |
| Cursor MCP config | `.cursor/mcp.json` |
| Agent skills | `.cursor/skills/21st-*` |

Skills installed:

- `21st-cli-use` — find / install / generate with the CLI
- `21st-ai` — 21st AI sketch generation
- `21st-registry` — publish / team registry
- `21st-design-sync` — design system sync

## One-time auth

1. Create an API key at [https://21st.dev/mcp](https://21st.dev/mcp)
2. Either:

```bash
cd app/frontend
npm run 21st:login
```

or set the env var (recommended for MCP):

```bash
export API_KEY_21ST="your-key-here"
```

3. Restart Cursor so MCP picks up `.cursor/mcp.json` and `API_KEY_21ST`.

## Useful commands

From `app/frontend`:

```bash
npm run 21st -- search "hero section"
npm run 21st -- search "spotlight card" --type c
npm run 21st -- get <id>
npm run 21st:add -- author/component-slug
npm run 21st -- generate "warm gold plum portfolio hero"
npm run 21st -- logo "react"
```

Or via npx from the repo root:

```bash
npx 21st search "bento grid"
npx 21st help
```

## Notes

- Do **not** commit API keys. Keep them in env vars or local login only.
- Component installs via `21st add` write files into the project using the 21st shadcn-compatible registry.
